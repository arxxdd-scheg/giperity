# giperity
ии модель, для классификации изображений.

## dataset 
данная модель основана на академическом дата сете, от image-net под названием mini imagenet

## свойства
модель gipermodel-0.1 и новее (до версии 1.0) имеет в своем дата сете более 60,000 изображений (600 изображений на один класс)

- веса модели

## gipermodel-(ver).json 
- классификация для модели
